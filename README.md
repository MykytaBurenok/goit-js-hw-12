# Image Search Gallery

A responsive image search application built with vanilla JavaScript and Vite.
The app uses the Pixabay API to search images by keyword, render results as a
gallery, and load more results with smooth scrolling.

## Live Demo

[live demo](https://mykytaburenok.github.io/Image-Search-Gallery/)

## Repository

[repository link](https://github.com/MykytaBurenok/Image-Search-Gallery)

## About the Project
This project started as a JavaScript assignment and was later refined into a portfolio-quality application.
The goal was not only to fetch and display API data, but also
to improve code structure, user feedback, and interaction flow.

## Features

- Search images by keyword.
- Fetch data from the Pixabay API.
- Render image cards dynamically.
- Load more results without reloading the page.
- Smooth scroll after loading new content.
- Show loading state while data is being fetched.
- Show error and empty-result messages with iziToast.

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES Modules)
- Vite
- Fetch API
- Pixabay API
- iziToast
- SimpleLightbox

## Key Skills Demonstrated?

- Working with asynchronous JavaScript using `async/await`
- Handling API requests and pagination
- Organizing logic into separate modules
- Updating the DOM dynamically based on API responses
- Managing loading, success, error, and empty states
- Improving user experience with smooth scrolling and UI feedback
